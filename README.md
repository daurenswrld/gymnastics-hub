# Gymnastics Hub - Базовая Структура

Это базовая структура проекта для верстки лендинга **Gymnastics Hub**. 
Проект включает в себя настроенные HTML, CSS, SCSS и JS файлы с премиальным дизайном.

## 📁 Структура проекта

- `index.html` — основная разметка страницы (SEO-friendly).
- `scss/` — исходные стили (модульная структура).
  - `abstracts/` — переменные и миксины.
  - `base/` — сброс стилей и типографика.
  - `style.scss` — главный файл стилей.
- `css/` — скомпилированные стили (уже готов `style.css`).
- `js/` — клиентские скрипты (плавный скролл, анимации).
- `assets/` — папка для изображений и шрифтов.

## 🚀 Как начать работу

1. Установите зависимости (если планируете использовать SCSS):
   ```bash
   npm install
   ```

2. Для запуска авто-компиляции SCSS в CSS используйте:
   ```bash
   npm run sass-watch
   ```

3. Для разовой компиляции и сжатия стилей:
   ```bash
   npm run sass-compile
   ```

4. Просто откройте `index.html` в браузере, чтобы увидеть результат.

## ✨ Особенности дизайна
- Современная типографика (шрифт **Outfit**).
- Эффекты при скролле (Reveal animations).
- Адаптивная (responsive) база.
- Использование градиентов и премиальной цветовой гаммы.
